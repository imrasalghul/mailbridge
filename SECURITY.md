# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We stan responsible disclosure. If you find anything that has the potential to harm masses, please email me at mailbridge@alghul.com (I get alerts for this one). 
Please give me at least seventy-two (72) hours before publicly making a GitHub issue to ensure that I have enough time to push out a fix before a vulnerability is discovered.
